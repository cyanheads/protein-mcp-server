# 0.2.1 — 2026-07-03

## Added

- **`protein_analyze_collection` / `protein_search_structures`** — nested cross-tab facet child dimensions now carry a `truncated` boolean when their bucket list is capped by `bucket_limit`; `renderFacets` appends `(truncated)` to the nested line in `content[]`. ([#13](https://github.com/cyanheads/protein-mcp-server/issues/13))

## Fixed

- **`protein_track_ligands` `binding_site`** — now caps returned instances to `limit` and emits a truncation notice, instead of returning every instance regardless of `limit`. ([#9](https://github.com/cyanheads/protein-mcp-server/issues/9))
- **Declared recovery hints** — `errors[].recovery` now reaches the wire (`data.recovery.hint`, mirrored into `content[]`) at throw sites across `protein_search_structures`, `protein_get_annotations`, `protein_get_structure`, `protein_find_similar`, `protein_track_ligands`, and `protein_analyze_collection`. ([#10](https://github.com/cyanheads/protein-mcp-server/issues/10))
